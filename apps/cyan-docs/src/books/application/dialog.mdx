# &lt;dialog&gt;

A small set of utilities and styles for native HTML `<dialog>` usage. These styles were migrated from the legacy design system and mapped to the v4 tokens.

## Usage

Basic structure:

```html
<dialog open>
  <div class="header">
    <h3>Dialog title</h3>
  </div>
  <div class="content">Content</div>
</dialog>
```

<DialogDemo />


The CSS lives in the `packages/cyan-css/src/components/dialog.css` file and uses the following tokens:

| Name | Default / Notes |
| --- | --- |
| `--color-surface-1` | Dialog surface background |
| `--cn-border-radius-large` | Dialog corner radius |
| `--shadow-elevation-2` | Dialog shadow/elevation |
| `--cn-gap` | Internal spacing |
| `--cn-border-width` | Border width (fallback `1px` used) |
| `--color-border` | Border color used for header divider |
| `--cn-font-family` | Font family for headings |
| `--cn-font-size-heading-4` | Heading size (fallback provided) |
| `--background-dialog-backdrop` | Backdrop color (new token)
