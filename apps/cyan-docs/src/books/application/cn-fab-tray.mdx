<h1>FAB Tray</h1>
<p>The FAB tray is a fixed-position container for floating action buttons. It is intended to hold primary actions that should be easily accessible to the user.</p>

<h2>Usage</h2>
<p>To use the FAB tray, add the single `nav#fab-tray` element to your page. There can only be one global FAB tray; place your floating action buttons inside this element.</p>

<figure class="demo">
  <nav id="fab-tray">
    {/* Add FABs here */}
  </nav>
  <FabTrayDemo />
</figure>

```html
<nav id="fab-tray">
  {/* Add FABs here */}
</nav>
```

<h2>Responsive Behavior</h2>

<h3>Rail Integration</h3>
<p>The FAB tray automatically adjusts its position when the rail navigation is present:</p>

- **Desktop (>620px):** Rail appears as left sidebar - FAB tray position unchanged
- **Mobile (â‰¤620px):** Rail appears as bottom bar - FAB tray moves up by rail height (72px)

This ensures the FAB tray is always accessible and not covered by the bottom navigation rail on mobile devices.
