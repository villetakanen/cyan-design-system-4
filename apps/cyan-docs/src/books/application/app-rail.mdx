# App Rail

App Rail is the primary navigation container for the application. It is identified by the `nav#rail` selector.

The App Rail is a html `nav` element, containing anchor links, with a `cn-navigation-icon` element for each link. See [cn-navigation-icon](/custom-elements/cn-navigation-icon)'s design book for more information.

> ## BREAKING CHANGE 
>
> We no longer provide a separate container for mobile (bar) and desktop (rail) primary navigation. Primary navigation container of 4.0.0 and beyond is `nav#rail`, and it will adjust size and position in relation to the viewport.
>
> Items available only on mobile or desktop can be conditionally rendered based on the viewport size by using the atomic CSS classes `.sm-hidden`, `.md-hidden`, and `.lg-hidden`.

## Layout and Spacing

The following CSS variables are available for customizing the layout and spacing of the app rail.

| Variable | Default value | Description |
|---|---|---|
| `--cn-width-rail` | `calc(var(--cn-grid) * 10)` (80px) | The width of the rail. |
| `--cn-margin-for-rail` | `var(--cn-width-rail)` | The margin applied to `main` and `cn-app-bar` to make space for the rail. |
| `--cn-padding-top-rail` | `8px` | The padding at the top of the rail. |
| `z-index-rail` | `2000` | The z-index of the rail. |
