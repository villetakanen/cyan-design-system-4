# `<cn-loader>`

The `<cn-loader>` component displays a loading indicator with a spinning dual-ring animation and an icon in the center. It provides visual feedback when content is loading or processing.

## Usage

### Properties / Attributes

| Name     | Attribute | Type    | Default | Description                                                                                                |
|----------|-----------|---------|---------|------------------------------------------------------------------------------------------------------------|
| `noun`   | `noun`    | String  | `'fox'` | The noun/name of the icon to display in the center of the loader. Uses the `<cn-icon>` component.         |
| `inline` | `inline`  | Boolean | `false` | When true, the loader is smaller and optimized for use inside buttons or other navigation elements.        |

### HTML

```html
<!-- Default loader with fox icon -->
<cn-loader></cn-loader>

<!-- Loader with custom icon -->
<cn-loader noun="star"></cn-loader>

<!-- Small loader for navigation -->
<cn-loader inline></cn-loader>

<!-- Custom icon for navigation -->
<cn-loader noun="cat" inline></cn-loader>
```

## Features

- **Dual-Ring Animation**: Rotating dual-ring border provides smooth loading indication
- **Static Center Icon**: Center icon provides visual context without distraction
- **Inline Variant**: Smaller size optimized for navigation elements
- **Customizable Icon**: Center icon can be changed using the `noun` property
- **Semantic Loading**: Clear visual indication of loading state
- **CSS Custom Properties**: Configurable via design system tokens
- **Smooth Animation**: Clean rotating ring animation

## Visual Behavior

### Default State
- Displays a spinning dual-ring border
- Center icon rotates slowly in the opposite direction
- Standard size suitable for content areas

### Inline State (`inline`)
- Smaller dimensions matching `--cn-line` height
- Same animations but scaled appropriately
- Suitable for buttons and navigation elements

### Animation Details
- **Dual ring**: Rotates 360Â° in 1.2 seconds (linear)
- **Center icon**: Static, provides visual context
- Icon has 44% opacity for subtle appearance
- Ring has 72% opacity for prominence

## Dependencies

This component requires:
- `cn-icon` component for displaying the center icon
- Proper icon SVG files in the `/icons/` directory
- **Design system tokens** for sizing and colors

## CSS Custom Properties

The component uses these design system tokens:

- `--cn-loader-size`: Size of the loader (default state)
- `--cn-loader-line-width`: Width of the spinning ring border
- `--cn-loader-color`: Color of both the ring and center icon
- `--cn-line`: Base line height unit (used for navigation variant)

### Default Token Values
- `--cn-loader-size`: 48px (calc(var(--cn-grid) * 12))
- `--cn-loader-line-width`: 2px (calc(var(--cn-grid) / 2))
- `--cn-loader-color`: Uses primary color palette (light-dark adaptive)

## Accessibility

- Uses semantic loading indication
- Icon provides context for the loading state
- Animations follow system motion preferences
- Color contrast meets accessibility standards
