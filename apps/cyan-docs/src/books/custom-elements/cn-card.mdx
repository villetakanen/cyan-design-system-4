# Cards

A `cn-card` is a fixed container for a piece of content. It can be used to display a small amount of information, or to highlight a piece of content.

<CardFlexDemo />

## Basic Usage

```html
<cn-card title="Card Title">
  Card content goes here.
</cn-card>
```

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `elevation` | `number` | `1` | Card elevation level (0-4) |
| `cover` | `string` | `undefined` | URL for the cover image |
| `srcset` | `string` | `undefined` | Responsive image sources for the cover image |
| `sizes` | `string` | `undefined` | Media conditions for selecting the appropriate image from srcset |
| `noun` | `string` | `''` | Icon name to display |
| `title` | `string` | `''` | Card title |
| `href` | `string` | `''` | Link URL for the card or title |
| `notify` | `boolean` | `false` | Shows notification indicator |
| `alert` | `boolean` | `false` | Shows alert indicator |

## Slots

| Slot | Description |
|------|-------------|
| default | Any content in the default slot will be rendered in the card body. The card body should be used sparingly for extract and synopsis purposes - and should consist of one or more paragraphs of text |
| `actions` | The actions to display at the bottom of the card |

## Elevation

The cards support 5 levels of elevation (0-4). The default elevation of a card is 1.

```html
<cn-card elevation="0" title="Flat Card">Bordered card with no shadow</cn-card>
<cn-card elevation="1" title="Default Card">Default elevation with subtle shadow</cn-card>
<cn-card elevation="2" title="Raised Card">Higher elevation with medium shadow</cn-card>
<cn-card elevation="3" title="Floating Card">High elevation with prominent shadow</cn-card>
<cn-card elevation="4" title="Top Card">Highest elevation with maximum shadow</cn-card>
```

## UI States

The card supports 3 UI states: `default`, `notify`, and `alert`.

```html
<cn-card title="Default Card">
  Normal card state
</cn-card>

<cn-card title="Notification Card" notify>
  This card has a blue notification indicator in the top-right corner.
</cn-card>

<cn-card title="Alert Card" alert>
  This card has a red alert indicator in the top-right corner.
</cn-card>
```

## CSS Custom Properties

The component uses CSS custom properties (tokens) for styling:

| Property | Default | Description |
|----------|---------|-------------|
| `--cn-card-background` | `light-dark(var(--chroma-primary-99), var(--chroma-primary-10))` | Card background color |
| `--cn-card-box-shadow` | `0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1)` | Card shadow |
| `--cn-border-radius-card` | `var(--cn-border-radius-large, 16px)` | Card border radius |

### Card Typography Tokens

| Property | Default | Description |
|----------|---------|-------------|
| `--cn-font-weight-headline-card` | `400` | Card title font weight |
| `--cn-font-size-headline-card` | `var(--cn-font-size-heading-4)` | Card title font size |
| `--cn-line-height-headline-card` | `var(--cn-line-height-heading-4)` | Card title line height |
| `--cn-letter-spacing-headline-card` | `var(--cn-letter-spacing-heading-4)` | Card title letter spacing |

## Examples

### Basic Card
```html
<cn-card title="Basic Card">
  This is the content of the card.
</cn-card>
```

### Card with Cover Image
```html
<cn-card 
  title="Beautiful Landscape"

  noun="camera">

</cn-card>
```

### Card with Responsive Images (srcset)
```html
<cn-card 
  title="Optimized Image" 
  cover="https://example.com/image-800.webp"
  srcset="https://example.com/image-400.webp 400w, https://example.com/image-800.webp 800w, https://example.com/image-1600.webp 1600w"
  sizes="(max-width: 768px) 100vw, 800px"
  noun="camera">
  This card uses responsive images for optimal performance.
</cn-card>
```

<CardSrcsetDemo />

When using CDN-optimized images (like Netlify Image CDN), you can generate the srcset programmatically:

```javascript
import { netlifyImage, generateSrcset } from '@utils/images/netlifyImage';

const firebaseUrl = 'https://example.com/original-image.jpg';
const optimized = netlifyImage(firebaseUrl, { 
  width: 800, 
  format: 'webp', 
  quality: 85 
});
const srcset = generateSrcset(firebaseUrl, [400, 800, 1600], { 
  format: 'webp' 
});
```

Then use in your component:

```html
<cn-card 
  title="CDN Optimized" 
  cover={optimized}
  srcset={srcset}
  sizes="(max-width: 768px) 100vw, 800px">
  Content here.
</cn-card>
```

### Card with Actions
```html
<cn-card title="Product Card">
  This is a product description with action buttons.
  <div class="toolbar border border-radius" slot="actions">
    <button>Primary Action</button>
    <button class="text">
      <cn-icon noun="delete"></cn-icon>
    </button>
    <button class="call-to-action">
      <cn-icon noun="add"></cn-icon>
    </button>
  </div>
</cn-card>
```

### Linked Card
```html
<cn-card 
  title="Article Title" 
  href="/articles/example"
  cover="https://picsum.photos/400/225"
  noun="books">
  Click anywhere on this card to read the full article.
</cn-card>
```

### Card with Long Title
```html
<cn-card 
  title="101 with ellipsis in the title, which is long enough to cut off"
  noun="fox">
  <p>Card titles automatically truncate to 2 lines with ellipsis.</p>
</cn-card>
```
