## Usage

### Tokens


| Name | Description | Default |
|------|-------------|---------|
| `--background-toggle-button` | Background color of the toggle button | --chroma-primary-40 / --chroma-surface-70 |
| `--background-toggle-button-off` | Background color of the toggle button when off | --chroma-surface-30 / --chroma-surface-80 |
| `--color-on-toggle-button` | Color of the toggle button when on | --chroma-primary-70 / --chroma-surface-30 |
| `--color-on-toggle-button-off` | Color of the toggle button when off | --chroma-surface-50 / --chroma-surface-70 |

### Events

The toggle button here logs all form component events to the console.
