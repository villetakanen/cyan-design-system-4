# cn-sortable-list

A sortable list component that allows reordering of items using drag and drop.

## Demo

Here is a live demo of the component. It will automatically adapt to light and dark modes.

  </div>
</div>


## API

The component's properties and attributes.

| Property | Type | Default | Description |
|---|---|---|---|
| `items` | `CnListItem[]` | `[]` | An array of objects, where each object has a `key` and `title` property. |

The `CnListItem` interface has the following structure:

```typescript
export interface CnListItem {
  key: string;
  title: string;
}
```

## Usage

To use the `cn-sortable-list` component, you need to set its `items` property with an array of `CnListItem` objects.

```html
<cn-sortable-list id="my-list"></cn-sortable-list>

<script>
  const myList = document.getElementById('my-list');
  myList.items = [
    { key: '1', title: 'Item 1' },
    { key: '2', title: 'Item 2' },
    { key: '3', title: 'Item 3' },
  ];

  myList.addEventListener('items-changed', (event) => {
    console.log('New item order:', event.detail.items);
  });
</script>
```

## Guidelines

Best practices and accessibility considerations.

### Best Practices
- Use a unique `key` for each item in the list.
- Provide a descriptive `title` for each item.

### Accessibility
- The component uses `draggable="true"` to enable drag and drop functionality.
- The `cn-icon` with `noun="drag"` provides a visual indicator for the drag handle.
