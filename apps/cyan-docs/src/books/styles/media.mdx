## Poster

The `.poster` utility creates a large, hero or poster like container 
for an image inside a `<section>`, `<article>`, or `<figure>` element. 
It is designed to make the image stand out and provide a visually appealing presentation.

### Example


<section class="surface elevation-1">
  <img class="poster" src="https://picsum.photos/800/400" alt="Description of the image" />
</section>


```html
<section class="surface elevation-1">
  <img class="poster" src="https://picsum.photos/800/800" alt="Description of the image" />
</section>
```

</article>

## Aspect ratio

### w-16-9

The `.w-16-9` utility class applies a 16:9 aspect ratio to an element, making it ideal for video and wide images.

<figure class="surface elevation-1">
  <img class="poster w-16-9" src="https://picsum.photos/800/450" alt="Description of the image" />
  <figcaption>This is a 16:9 aspect ratio image.</figcaption>
</figure>

```html
<figure class="surface elevation-1">
  <img class="poster w-16-9" src="https://picsum.photos/800/450" alt="Description of the image" />
</figure>
```

</article>

## The `.icon` class

Sometimes you just want to use a plain `<img>` tag, but have it behave like a `cn-icon`. For this, you can use the `.icon` class. It will make the image scale with the text, and you can use the size modifiers to control the size of the icon.
