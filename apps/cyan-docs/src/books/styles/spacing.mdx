# Spacing Atomics

A comprehensive set of atomic classes for controlling padding, margin, and sizing. These utilities are based on the Cyan Design System's 8px baseline grid (`--cn-grid`) and provide consistent spacing throughout your application.

## Padding Classes

Control padding using these utility classes. All values are based on `--cn-grid` (8px).

### All Sides
| Class | CSS | Description |
| --- | --- | --- |
| `.p-1` | `padding: var(--cn-grid)` | 8px padding on all sides |
| `.p-2` | `padding: calc(var(--cn-grid) * 2)` | 16px padding on all sides |

### Horizontal Padding
| Class | CSS | Description |
| --- | --- | --- |
| `.px-1` | `padding-left: var(--cn-grid); padding-right: var(--cn-grid)` | 8px horizontal padding |
| `.px-2` | `padding-left: calc(var(--cn-grid) * 2); padding-right: calc(var(--cn-grid) * 2)` | 16px horizontal padding |

### Vertical Padding
| Class | CSS | Description |
| --- | --- | --- |
| `.py-1` | `padding-top: var(--cn-grid); padding-bottom: var(--cn-grid)` | 8px vertical padding |
| `.py-2` | `padding-top: calc(var(--cn-grid) * 2); padding-bottom: calc(var(--cn-grid) * 2)` | 16px vertical padding |

### Individual Sides
| Class | CSS | Description |
| --- | --- | --- |
| `.pt-1` | `padding-top: var(--cn-grid)` | 8px top padding |
| `.pt-2` | `padding-top: calc(var(--cn-grid) * 2)` | 16px top padding |
| `.pr-1` | `padding-right: var(--cn-grid)` | 8px right padding |
| `.pr-2` | `padding-right: calc(var(--cn-grid) * 2)` | 16px right padding |
| `.pb-1` | `padding-bottom: var(--cn-grid)` | 8px bottom padding |
| `.pb-2` | `padding-bottom: calc(var(--cn-grid) * 2)` | 16px bottom padding |
| `.pl-1` | `padding-left: var(--cn-grid)` | 8px left padding |
| `.pl-2` | `padding-left: calc(var(--cn-grid) * 2)` | 16px left padding |

```html
<div class="p-1 surface">8px padding on all sides</div>
<div class="px-2 py-1 surface">16px horizontal, 8px vertical padding</div>
<div class="pt-2 pr-1 pb-1 pl-2 surface">Individual side padding</div>
```

## Size Classes

Control element sizing using container query units for responsive design.

| Class | CSS | Description |
| --- | --- | --- |
| `.full-w` | `width: 100cqw` | Full container query width |
| `.full-h` | `height: 100cqh` | Full container query height |

```html
<div class="full-w surface">Full container width</div>
<div style="height: 200px; container-type: size;">
  <div class="full-h surface">Full container height</div>
</div>
```

## Design System Integration

These spacing atomics are designed to work seamlessly with the Cyan Design System's grid-based approach:

- **Consistent**: All spacing is based on the 8px baseline grid (`--cn-grid`)
- **Scalable**: Values are calculated using CSS custom properties
- **Responsive**: Container query units (`cqw`, `cqh`) adapt to container size
- **Predictable**: Simple naming convention follows Tailwind CSS patterns

For more information about the underlying grid system, see [Units and Grid](/books/principles/units-and-grid).
