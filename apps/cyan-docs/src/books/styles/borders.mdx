 # Borders

 We provide some simple Tailwind-inspired `atomics` for borders. 

 | Class Name | Description |
 | ---------- | ----------- |
 | `border` | Apply a border to an element. |
 | `border-t` | Apply a top border to an element. |
 | `border-r` | Apply a right border to an element. |
 | `border-b` | Apply a bottom border to an element. |
 | `border-l` | Apply a left border to an element. |

 ## Examples

## Border Radius

Control the roundness of corners with the `radius-*` classes.

| Class Name | Description |
| --- | --- |
| `radius-s` | Small border radius |
| `radius-m` | Medium border radius |
| `radius-l` | Large border radius |
| `radius-xl` | Extra large border radius |
| `radius-round` | Circular border radius |
