# `<fieldset>`

The `<fieldset>` element groups related form controls with consistent border-radius and border styling that matches the design system's visual language. The `<legend>` element provides a caption for the fieldset group.

## Basic Fieldset

Fieldsets receive design system styling with rounded corners, themed borders, and proper spacing. The legend element is styled to integrate seamlessly with the fieldset border.

### Example

import FieldsetDemo from 'src/components/demo/FieldsetDemo.astro';

```html
<fieldset>
  <legend>Contact Information</legend>
  <label>
    Email
    <input type="email" name="email">
  </label>
  <label>
    Phone
    <input type="tel" name="phone">
  </label>
</fieldset>
```

Renders as:

<div class="grid two-col">
  <div class="light border border-radius p-1 surface">
    <h3>LIGHT</h3>
    <FieldsetDemo />
  </div>
  <div class="dark border border-radius p-1 surface">
    <h3>DARK</h3>
    <FieldsetDemo />
  </div>
</div>

## Design Tokens

The fieldset element uses the following design tokens:

```css
fieldset {
  border: 1px solid var(--color-border);
  border-radius: var(--cn-border-radius-medium);
  padding: var(--cn-grid);
  margin: 0;
  margin-bottom: var(--cn-grid);
}

legend {
  padding: 0 calc(var(--cn-grid) * 0.5);
  font-weight: var(--cn-font-weight-heading, bold);
  font-size: var(--cn-font-size-caption);
  color: var(--color-on-label);
}
```

**Key tokens:**
- `--color-border`: Border color (supports light/dark themes)
- `--cn-border-radius-medium`: Rounded corners (8px by default)
- `--cn-grid`: Spacing unit for padding and margins
- `--color-on-label`: Legend text color
- `--cn-font-weight-heading`: Legend font weight
- `--cn-font-size-caption`: Legend font size

## Accessibility

Fieldsets provide important semantic and accessibility benefits:

- **Semantic Grouping**: The `<fieldset>` element clearly identifies a group of related form controls
- **Legend**: The `<legend>` element provides a label for the entire group, which screen readers announce
- **Context**: Screen readers announce the legend when users navigate to any control within the fieldset
- **Best Practice**: Always use a `<legend>` when using a `<fieldset>` for accessibility

### Example with Good Accessibility

```html
<!-- Good: Clear legend for screen readers -->
<fieldset>
  <legend>Payment Method</legend>
  <label>
    <input type="radio" name="payment" value="card">
    Credit Card
  </label>
  <label>
    <input type="radio" name="payment" value="paypal">
    PayPal
  </label>
</fieldset>
```

Screen readers will announce: "Payment Method, group" when entering the fieldset, and "Credit Card, radio button, Payment Method group" for each control.
