# Shadows and Elevation

There are two ways to add shadows to elements: shadow utilities and elevation utilities. 

## Shadow Utilities

The `.shadow-[1...4]` utility classes can be used to add a material-design inspired shadow to an element, without changing its background color.

## Elevation Utilities

By default, all layout elements are transparent on top of application backgrounds. The
`.surface` and `.elevation-[1...4]` utility classes can be used to place content on 
'top of a surface, with a varying level of material-design inspired elevation from the 
background. These utilities change the background color and add a shadow. Shadow
depth increases with the difference of elevation.
