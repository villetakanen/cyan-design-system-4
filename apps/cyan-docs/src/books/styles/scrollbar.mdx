# Scrollbar

The design system provides a semi-transparent scrollbar style applied globally to all scrollable elements. The scrollbar features a transparent track with a visible, semi-transparent thumb that becomes more prominent on hover.

## Design

The scrollbar is designed to be:
- **Subtle**: Semi-transparent thumb (~50% opacity) on a transparent track
- **Discoverable**: Thumb becomes more visible on hover (~70% opacity)
- **Consistent**: Same styling across all scrollable elements
- **Theme-aware**: Adapts to light and dark modes

## Tokens

The following design tokens control the scrollbar appearance:

| Token | Default | Description |
|-------|---------|-------------|
| `--cn-scrollbar-width` | `8px` | Width of the scrollbar |
| `--cn-scrollbar-border-radius` | `4px` | Border radius of track and thumb |
| `--color-scrollbar-track` | `transparent` | Track background color |
| `--color-scrollbar-thumb` | semi-transparent | Thumb color (theme-aware) |
| `--color-scrollbar-thumb-hover` | semi-transparent | Thumb hover color (theme-aware) |

## Customization

Override the tokens to customize the scrollbar appearance:

```css
/* Make the scrollbar more visible */
:root {
  --color-scrollbar-thumb: rgba(128, 128, 128, 0.7);
  --color-scrollbar-thumb-hover: rgba(128, 128, 128, 0.9);
}

/* Wider scrollbar */
:root {
  --cn-scrollbar-width: 12px;
}
```

## Browser Support

The scrollbar styling works across modern browsers:
- **Chrome, Edge, Safari**: Uses `::-webkit-scrollbar` pseudo-elements
- **Firefox**: Uses `scrollbar-width` and `scrollbar-color` properties
