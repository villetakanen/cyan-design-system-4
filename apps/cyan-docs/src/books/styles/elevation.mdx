# Surfaces and Elevation

By default, all layout elements are transparent on top of application backgrounds. The
`.surface` and `.elevation-[1...4]` utility classes can be used to place content on 
top of a surface, with a varying level of material-design inspired elevation from the 
background.

## Secondary surface

The `.secondary` class can be used to create a secondary-colored surface. This is useful for sidebars, headers, and other UI elements that need to stand out from the main content area.
