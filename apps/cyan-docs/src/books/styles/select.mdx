# `<select>`

Cyan provides styling for the native `<select>` element, designed to work seamlessly across different layout contexts including standalone labels, flex containers, toolbars, and fieldsets.

## Basic Select

Select elements support standard states including disabled and invalid for validation feedback.

### Example

import SelectDemo from 'src/components/demo/SelectDemo.astro';

```html
<label>
  Text
  <select>
    <option>Option 1</option>
    <option>Option 2</option>
    <option>Option 3</option>
  </select>
</label>
<label>
  Disabled
  <select disabled>
    <option>Disabled</option>
  </select>
</label>
<label>
  Invalid
  <select required>
    <option value="">Please select an option</option>
    <option>Option 1</option>
    <option>Option 2</option>
  </select>
</label>
```

Renders as:

<div class="grid two-col">
  <div class="light border border-radius p-1 surface">
    <h3>LIGHT</h3>
    <SelectDemo />
  </div>
  <div class="dark border border-radius p-1 surface">
    <h3>DARK</h3>
    <SelectDemo />
  </div>
</div>

## Layout Behavior

Select elements inside labels are designed to fill their container width by default (`width: 100%`), ensuring consistent behavior across different layout contexts.

### Standalone Labels

In standalone labels, the label container itself sizes to `width: auto` while the select fills the available space:

```html
<label>
  Choose a category
  <select>
    <option>Books</option>
    <option>Games</option>
    <option>Movies</option>
  </select>
</label>
```

## Technical Details

The select element behavior is controlled by these CSS rules:

**Label container sizing:**
```css
label:has(select) {
  width: auto;        /* Label sizes to content */
  flex-grow: 0;       /* Doesn't grow in flex containers */
}
```

**Select element sizing:**
```css
label select {
  min-width: auto;    /* Override browser defaults */
  width: 100%;        /* Fill label container */
}
```

This ensures:
- Labels with selects size naturally to their content (`width: auto`)
- Labels don't grow to fill flex containers (`flex-grow: 0`)
- Selects fill their label container (`width: 100%`)
- Browser minimum widths are overridden (`min-width: auto`)
