Modal Rail Hiding Test

This page tests whether the sm-hidden class properly hides the rail navigation.

Test Instructions:

  1. Resize the browser window to see how the rail behaves at different sizes
  2. At ≤620px: Rail should show as bottom bar
  3. At >620px: Rail should show as left sidebar
  4. Click "Toggle Modal" to apply/remove sm-hidden class to the rail
  5. When sm-hidden is applied and viewport ≤620px: Rail should be hidden
  6. FAB Tray Testing: Notice how the blue FAB button moves up when rail is visible on mobile

Expected FAB behavior:

  • ≤620px + rail visible: FAB positioned 96px from bottom (24px + 72px rail height)
  • ≤620px + rail hidden: FAB positioned 24px from bottom
  • >620px: FAB always positioned 24px from bottom (rail is sidebar)
